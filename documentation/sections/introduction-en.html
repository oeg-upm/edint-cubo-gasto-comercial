Introduction back to ToC

Commercial expenditure data are fundamental for socio-economic analysis, territorial planning, and the study of consumption patterns within a given territory. However, these data are often published in heterogeneous formats with little explicit semantics, which hinders their interoperability and reusability. In this context, the W3C Data Cube vocabulary provides an adequate framework for representing multidimensional statistical data in a structured and reusable manner. Moreover, the reuse of consolidated vocabularies and code lists allows for a more precise description of relevant dimensions such as territory, reference period, economic activity sector, or nationality. The objective of this vocabulary is to define the semantic structure of the commercial expenditure data cube, specifying the necessary dimensions and measures to represent commercial expenditure information in an interoperable way and in accordance with open standards.
